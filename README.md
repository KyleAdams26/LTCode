Liver Transplant Rejection Model Code

This repo holds all the MATLAB code used in Bruner et. al liver transplant rejection model.

Note: Some files will be updated asap to be reusable for other models. These include SobolMain, getSamplesDesiredDist, getSamplesSobol, and sobolAnalysis. Other files should be reusable as of 11/26/24. If there are any errors or questions, please email kyleadams2600@gmail.com.

As of 3/11/25 most of the files have more comments to assist the reader. We hope to make the code more efficient in the future. If there are any errors, questions, or suggestions, please email kyleadams2600@gmail.com.
